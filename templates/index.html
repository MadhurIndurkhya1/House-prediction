    Model Architecture & Insights

    This model is built using a scikit-learn machine learning pipeline utilizing a regularized Linear Regression (Ridge) algorithm to predict property prices in Bengaluru.

    Pipeline Workflow

    Input features are mapped through a ColumnTransformer. Categorical locations are One-Hot Encoded while numeric inputs are standardized via a StandardScaler before fitting the estimator.

    Outlier Preprocessing

    Cleaned dataset outliers were pruned by removing records with area smaller than 300 sq. ft. per BHK and filtering out local price-per-sqft anomalies outside one standard deviation (±1σ).
